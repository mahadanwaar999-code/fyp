## Audio Specifications

- **Format:** MP3
- **Duration:** Approximately 5-7 minutes per section
- **Quality:** At least 128 kbps
- **Sample Rate:** 44.1 kHz recommended
- **Channels:** Mono or Stereo

## File Naming Convention

IMPORTANT: Files must be named exactly as shown:
- Section 1: `section1.mp3`
- Section 2: `section2.mp3`
- Section 3: `section3.mp3`
- Section 4: `section4.mp3`

## Testing Your Audio

After adding the files:
1. Navigate to the Student Panel
2. Click "Take Practice Test"
3. Select "Individual Section"
4. Choose "IELTS Academic" or "IELTS General Training"
5. Select "Listening"
6. Click "Start Test"
7. The audio player should load and play your audio file

## Troubleshooting

**Audio not playing?**
- Check that files are in the correct folder: `/public/audio/listening/`
- Verify file names match exactly (case-sensitive)
- Ensure files are in MP3 format
- Check browser console for error messages
- Try a different browser (Chrome, Firefox, Safari, Edge)

**Audio cuts off early?**
- Check the audio file duration
- Ensure the file is not corrupted
- Re-export/convert the audio file

**No sound?**
- Check your device volume
- Check browser permissions for audio playback
- Try unmuting the tab in your browser
